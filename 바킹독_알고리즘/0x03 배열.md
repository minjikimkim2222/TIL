1.배열 초기화 팁
==============
배열 전체를 특정 값으로 초기화하는 3가지 방법
<pre>
<code>
int a[21];
int b[21][21];

// 1. memset -> 오류 많아서 비추천!!
memset(a, 0, sizeof(a));
memset(b, 0, sizeof(b));

// 2. for문 -> 실수 없고 무난함
for (int i = 0; i < 21; i++)
   a[i] = 0;
   
for (int i = 0; i < 21; i++)
    for(int j=0; j < 21; j++)
      b[i][j] = 0;
      
// 3. fill 함수 -> 익숙해진다면, 가장 추천
fill(a, a + 11, 0);

for (int i = 0; i < 21; i++)
  fill(b[i], b[i] + 21, 0);
</pre>
